# Packer 

This code can be used to spin up an EC2 instance, provison Apache on Amazon Linux OS and package it as an Amazon Machine Image (AMI).
The pipeline gets triggered when there is a `push` on the master branch.

## Add AWS Credentials

```
{
"aws_access_key":"",
"aws_secret_key":"",
"token":"",
"ssh_username":""
}

```
